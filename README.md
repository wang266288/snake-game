# Rust Snake Game
一个使用 Rust 和 macroquad 游戏引擎编写的贪吃蛇游戏，支持AI 自动游玩和记录本地最高分。
## 功能特性
### 快速开始
克隆仓库
```
git clone <repository-url>
cd snake-game
```
运行游戏
```
cargo run
```
操作说明
↑ ↓ ← → — 控制蛇方向
A — 切换 AI 模式（开/关）
R — 游戏结束后重新开始
### AI 算法简述
AI 采用“**BFS 优先**+**安全兜底**”策略：
**BFS 寻路**：在将蛇尾视为可通行的网格上搜索到达食物的最短路径。
**安全性验证**：检查候选方向是否安全（不撞墙/自身），并计算该移动后可达区域面积。
**评分回退**：若 BFS 路径不安全，则对四个方向进行综合评分（可达面积、距食物距离、距蛇尾距离），选择最高分方向。
该设计在保证安全的前提下尽可能高效地接近食物。
